# Instalação corrigida para Vercel

Este projeto foi configurado para usar Node.js 24 e pnpm 10.28.1.

Na Vercel:

1. Mantenha **Node.js Version: 24.x**.
2. Em **Install Command**, desative qualquer Override manual.
3. Faça um novo deploy sem cache.

O arquivo `vercel.json` já instala o pnpm e executa o build automaticamente.
